Week 8 — Introduction to Python

Python keeps every idea from C++ but is simpler to write. Edit and run each example below.

BIG PICTURE

Same ideas, simpler language

You already know how to think like a programmer. Python keeps variables, expressions, conditionals, loops and functions — but with simpler syntax: no type declarations, no semicolons, no braces. Indentation (the spaces at the start of a line) defines the structure instead.

PYTHON

  • f"...{power}..." is an f-string: it drops the value of power into the text.
A KEY DIFFERENCE FROM C++

Division in Python

In Python, / always gives a real answer (no integer-division trap). Use // for whole-number division. Run it and see.

No experience needed

This course assumes you have never written code before. We start from the very beginning. You do not need to be "good with computers" — just willing to follow clear steps, like working through a maths problem. Mistakes are normal; finding and fixing them is part of learning.

CONCEPT

What exactly is a program?

A computer does nothing on its own — it only follows instructions. A program is a precise, step-by-step list of instructions that turns some input into a useful result. You must describe the solution in such exact detail that a machine, which has no common sense, can carry it out without guessing.

Instructions

Each line is a command the computer obeys in order, top to bottom.

Input → Process → Output

Almost every program reads data in, transforms it, and produces a result out.

No guessing

The computer does exactly what you write — not what you meant. Precision is everything.

C++

  • cout << "..." prints text to the screen; endl moves to a new line.
